# Track 18: Stem-Aware Shader Visuals (Reactive Dashboard)

## Overview
Transform the UI into a high-performance visualizer. Use the extracted stems (from Track 15) and CNN insights (from Track 14) to drive complex GLSL shaders in the Vue dashboard, creating a visual experience that is perfectly synced to the music's structure.

## Requirements
- Integrate a WebGL/Three.js shader canvas into the Vue UI.
- Map audio features (Vocal intensity, Drum transients) to shader uniforms.
- Create "state-aware" visuals that change based on AI intents (e.g., "Tension" vs. "Drop").
