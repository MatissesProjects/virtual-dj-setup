# Product Definition: Virtual DJ Setup

## Vision
A real-time, AI-collaborative virtual DJ system that bridges high-performance audio processing (C#) with advanced machine learning (Python). The system is designed for live broadcast (e.g., Twitch) with reactive and predictive mixing capabilities.

## Core Features
- **Dual-Stream Audio Capture:** WASAPI Loopback capture of browser/source streams.
- **High-Performance DSP Chain:** Real-time EQ, Compression, Limiting, and M/S processing.
- **AI Intelligence Layer:** Predictive state management and intent-driven control.
- **Manual Override (Shadow Mode):** Seamless transition between AI and human control, with the AI learning from human actions.
- **Multi-Modal Feedback:** Integration with crowd metrics (e.g., Twitch chat) to drive the "vibe".
- **Hardware Integration:** MIDI support for physical control surfaces.

## Target Audience
- Streamers/DJs looking for AI augmentation.
- Developers interested in real-time DSP and Reinforcement Learning.

## Success Metrics
- Microsecond latency in the DSP chain.
- Accurate predictive mixing (e.g., triggering transitions before transients).
- High user engagement (measured via crowd vibe score).
